Administration & Setup

/setmodroleArgs: roleThe "Master Key" command. Authorizes a specific role to use moderator commands.
/log_setupArgs: channel, eventsConnects the audit logging system to a channel to track deleted messages, bans, and edits.
/setup_welcomeArgs: channel, message, image (optional)Initializes the visual welcomer with customizable text and backgrounds.
/rolepanelArgs: channel, titleCreates a reaction role menu allowing users to self-assign roles.

Moderation

/banArgs: user, reasonPermanently bans a user from the server.
/kickArgs: user, reasonRemoves a user from the server but allows them to rejoin.
/mute (Timeout)Args: user, duration, reasonTimes out a user, preventing them from typing or joining voice channels.
/warnArgs: user, reasonIssues a formal warning to a user which is logged in their history.
/tempbanArgs: user, duration (e.g. 24h), reasonBans a user and automatically unbans them after the set duration.
/historyArgs: userDisplays a comprehensive list of past infractions for a specific user.
/lockArgs: channel (optional)Freezes chat in the current or specified channel so members cannot type.
/nukeArgs: noneClones and deletes the current channel to wipe all history instantly. Irreversible.

Support Tickets

/setup_ticketArgs: channel, categoryDeploys the "Create Ticket" panel to a channel.
/configure_optionArgs: label, roleLinks specific dropdown options to roles (e.g., "Billing" alerts Admin).
/claimArgs: noneAssigns the current ticket to you, letting other staff know you are handling it.

Leveling & XP

/rankArgs: user (optional)Generates a visual rank card showing your level, XP, and progress bar.
/leaderboardArgs: noneDisplays the top 10 most active members in the server.
/reward addArgs: level, roleConfigures the bot to automatically give a Role when a user hits a specific Level.
/xp_multi_channelArgs: channel, multiplierSets a channel as a bonus zone (e.g., 2x XP) to encourage activity there.

Utilities

/remindArgs: time, messageSets a personal DM reminder (e.g., /remind 1h Check emails).
/timer_startArgs: duration, titleStarts a public countdown timer in the channel.
/tag createArgs: name, contentCreates a custom snippet/macro that can be recalled later using /tag use.
/sticky setArgs: messagePins a message to the bottom of the chat view, reappearing after every new message.
/afkArgs: messageSets your status to AFK. Users who ping you will be told you are away.
/whoisArgs: userDisplays deep user analysis including join dates, roles, and account age.
/certifyArgs: image_urlFun utility that stamps an image with a "Certified" watermark.

Giveaways & Events

/gstartArgs: duration, winners, prize, requirementsStarts a giveaway. You can set requirements like "Must have Level 5 Role".
/grerollArgs: message_idPicks a new winner for an ended giveaway if the previous winner was invalid.
/suggestArgs: ideaPosts your suggestion to the server's feedback channel with Up/Down voting buttons.
