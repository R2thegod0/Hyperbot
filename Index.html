Command Reference

Complete documentation for all modules.

Administration & Setup

/setmodrole Set the role allowed to use moderator commands.
Args: role
/log_setup Configure logging channels.
Args: log_type (Mod/Server/Join/All), channel
/log_ignore Stop logging messages from a specific channel (e.g. spam).
Args: channel
/setup_welcome Set up welcome image cards.
Args: channel, background_url (optional)
/setup_stats Create locked voice channels for member counts.
/rolepanel Create a Reaction Role panel.
Args: title, description, mode (Toggle/Pick One)
/role_add Add a button to a role panel.
Args: message_id, role, label, emoji, style

Moderation

/ban Ban a user.
Args: user, reason
/tempban Ban a user temporarily.
Args: user, duration (e.g. 1d, 30m), reason
/kick Kick a user.
Args: user, reason
/mute Timeout a user.
Args: user, minutes, reason
/unmute Remove timeout.
Args: user
/warn Issue a warning.
Args: user, reason
/history View all mod actions against a user.
Args: user
/lock Lock current channel.
/nuke Clone and delete channel (Clears history).

Support Tickets

/setup_ticket Create the ticket panel interface.
/configure_option Link a dropdown option to a category/role.
Args: panel_id, option_name, role, category
/add Add user to ticket.
Args: user
/claim Assign ticket to yourself.
